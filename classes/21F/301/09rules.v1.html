Why formalize?

Formalization allows us to more carefully and precisely state our hypotheses and check them.

Phonological Rules

So far, we have used the term phonological rule synonymously with phonological process, phonological change, and phonological transformation.

From now on, we will use the term phonological rule to refer to our formal grammatical description.

A phonological rule has three parts: the what, the how, and the where. A formal rule says what changes, how it changes and the context in which the change takes place. Here is the general form of the rule: A → B / C _ D.

  • We read this rule as "A becomes B between C and D".
  • A is the target of the rule (the what)
  • B is the structural change of the rule (the how)
  • C _ D is the structural description of the rule (the where). The structural description is also called the rule's environment or context.

Examples

  1. Kerewe
  • h → p / m _
  • "h becomes p after m"
  1. Finnish
  • e → i / _ #
  • "e raises to i at the end of words"

Rules for Insertion and Deletion

  1. Insertion
  • ∅ → A / C _ D
  • "Insert A after C and before D"
  1. Deletion
  • A → ∅ / C _ D
  • "Delete A whenever it occurs between C and D"

Multiple targets: describing the target

Fijian. "Consonants delete word finally." How would we write this? One way is to just have a rule for each consonant.

  • ð → ∅ / _ #
  • r → ∅ / _ #
  • t→ ∅ / _ #
  • ...

Another way would be to put the consonants in curly braces like this.

  • {ð, r, t, ...} → ∅ / _ #

A better way however is to use the phonetic descriptive term in square brackets.

  • [consonant] → ∅ / _ #

A common abbreviation is to use capital C for [consonant].

  • C → ∅ / _ #

Paamese. "Vowels delete word finally." How would we write this?

  • [vowel] → ∅ / _ #
  • V → ∅ / _ #

Multiple targets: describing the change

Russian. Devoicing: "Stops devoice word finally." If we have a rule for each stop, we would write something like this.

  • g → k / _ #
  • d → t / _ #
  • b → p / _ #

We want to write this as a single rule. We can write the target of the rule as [stop]. But how do we write its structural change? The idea is that when [g] becomes [k] the only change is the voicing. The manner and place of articulation stay the same. The structural change should only refer to the phonetic properties that change. Conventionally, we write what it becomes.

  • [stop] → [voiceless] / _ #

This changes [g] to [k], [d] to [t], and [b] to [p] word-finally.

When phonological changes have multiple environments

Recall the analysis of the regular English plural. We posited two rules.

  1. Devoicing: "z becomes s after voiceless stops"
  2. Schwa Insertion: "a schwa is inserted between two sibilants"

Consider Devoicing. There are three voiceless stops in English [p, t, k]. So we could write three rules like this.

  • z → [voiceless] / p _
  • z → [voiceless] / t _
  • z → [voiceless] / k _

As before, we want to combine these into a single rule. Again, one way is to use curly braces.

  • z → [voiceless] / {p,t,k} _

A better way is to list the phonetic properties that picks out those elements.

  • z → [voiceless] / [voiceless,stop] _

Discussion

  1. In the rule for English Devoicing, we wrote [voiceless,stop] in the rule's environment. Is this list of phonetic properties interpreted conjunctively or disjunctively? For a list [X,Y] "conjunctively" means "X and Y" whereas "disjunctively" means "X or Y".
  2. In Russian Devoicing we wrote [stop] for the rule's target. This suggests that the rule applies to both voiced and voiceless stops. What happens when the rule "applies" to a voiceless stop?
