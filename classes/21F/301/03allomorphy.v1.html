Allomorphy is the phenomenon when the same morpheme exhibits distinct pronunciations.

The morphemes in the word 'allomorphy' are shown in the table below.

Morpheme Meaning
allo ‘different’
morph ‘form’
y ‘state of’
  • Allomorphy complicates the comparison and subtraction methods.
  • Allomorphy is the rule, not the exception.
  • Allomorphy raises questions about how our knowledge of language is organized, mentally.

The Question Allomorphy Raises

  • What long-term memory representation of the pronunciation of these morphemes do speakers have?
  • When there is no allomorphy, the answer is straightforward: what you see is what you get.

In Kerewe, the long-term memory representation of the pronunciation for ‘measure’ is plausibly [pima] because it is always pronounced [pima]. But what is the long-term memory representation of the pronunciation for ‘gasp’? Is it [hiima or piima]? Or both?

We call the long-term memory representation the underlying form.

Analysis #1: Phonological Hypothesis

  1. Kerewe speakers store ONE representation of the verbs in long-term memory.
  2. This form undergoes phonological changes depending on its phonological context.

In Kerewe, speakers store [pima] for ‘measure’ and [hiima] for ‘gasp’ and similarly for the other verbs. Also, Speakers change [h] to [p] after [m].

Analysis #2: Morphological Selection Hypothesis

  1. Kerewe speakers store the pronunciation of each allomorph in long-term memory.
  2. They select which one to use based on some combination of phonological or morphological factors.

In Kerewe, speakers store [pima] for ‘measure’ and store [hiima and piima] for ‘gasp’ and similarly for the other verbs. To pronounce a morpheme, if there is only one stored pronunciation then they use that pronunciation. When there are two stored forms, speakers select the pronunciation which begins with [p] whenever the 1st sg habitual prefix is used and select the pronunciation beginning with [h] everywhere else.

Summary

  1. When doing morphological analysis, the comparison and subtraction methods often reveal more than one pronunciation for morphemes.
  2. There are multiple hypotheses consistent with the data.
    • Phonological
    • Morphological Selection
  3. The Phonological analysis predicts that the allomorphic variation is systematic.
  4. The Morphological Selection analysis does not.
