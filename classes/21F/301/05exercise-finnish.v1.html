  1. The partitive singular suffix is alternates between [-a] and [-æ], whereas the nominative singular is a zero affix. It is predictable which allomorph occurs but the phonological environment is complicated. So that is an advanced topic that we can return to later. For now, let us examine the allomorphy in the root nouns.
